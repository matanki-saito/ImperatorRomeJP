本MODで使用しているフォントは、以下のフォントを加工したものです。
Hellenism-SemiBold      →「源流明朝」（https://github.com/ButTaiwan/genryu-font）
Hellenism-Kana-SemiBold →「刻明朝」（http://freefonts.jp/font-koku-min.html）

【ライセンス】
Hellenism-SemiBoldのライセンスは「SIL Open Font Licence 1.1」に従います。
Hellenism-Kana-SemiBoldのライセンスは「IPA Font License Agreement v1.0」に従います。
各ライセンスは、このファイルと同じフォルダにあるテキストファイルをご覧ください。

【IPAフォントへの置き換え方】（ライセンス上の必要性から記載しますが、置き換えを推奨するものではありません）
1. http://ipafont.ipa.go.jp/ipaexfont/download.html より、フォントをダウンロードしてください。
2. ダウンロードしたフォントを、このファイルと同じフォルダにコピーしてください。
3. fonts.fontファイルをテキストエディターで開き、Hellenism-Kana-SemiBold.ttfの部分（5カ所）をダウンロードしたファイルの名前に書き換えてください。
